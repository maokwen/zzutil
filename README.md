# 测试 UDP 收发

## Build

You need to have `cmake` and `mingw` installed.

### VS Code:

`Ctrl+Shift+B` to build.

`F5` to run.

see `build` directory for output.

### Command line:

```cmd
mkdir cmake_build
cd cmake_build
cmake .. -G "MinGW Makefiles"
mingw32-make .
```

see `build` directory for output.
